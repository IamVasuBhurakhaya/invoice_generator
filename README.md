# Invoice Generator App

## Project Description

The Invoice Generator App is a mobile application developed using Flutter designed to automate and simplify the process of creating and managing invoices. This project showcases a clean and functional UI for generating invoices, with essential features built using fundamental Flutter widgets and dependencies. The app provides a practical demonstration of Flutter concepts including forms, state management, and PDF generation.

### Flutter PDF Package

The app utilizes the `flutter_pdf` package to generate and manage PDF invoices.

## Project Objectives

### Elegant UI

The app features a modern and user-friendly interface designed to streamline the invoice generation process.

### Invoice Management

- **Invoice Creation**: Allows users to create invoices by entering customer details, item descriptions, and pricing.
- **PDF Generation**: Enables users to generate and download invoices in PDF format.
- **Invoice History**: Provides a history view of previously created invoices.

## Project Technologies

- **Language**: Dart
- **Dependency**: flutter_pdf
- **Development Tools**: Android Studio, VS Code, IntelliJ IDEA

## App Features

- **Invoice Creation**: Users can enter details such as customer information, items, and amounts to generate invoices.
- **PDF Export**: Invoices can be exported as PDFs for easy sharing and printing.
- **Invoice History**: Users can view a list of previously generated invoices.
- **Edit and Delete**: Allows users to edit or delete previously created invoices.
- **Customizable Templates**: Provides customizable templates for different types of invoices.
